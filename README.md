# <img src=".gitlab/ansible.png" alt="ansible" height="30"/> Ansible Roles

::include{file=.gitlab/badges.md}

Kolekcja ról Ansible do zarządzania konfiguracją serwerów Linux. Role wspierają wiele dystrybucji (Debian/Ubuntu, RHEL/AlmaLinux, Alpine) i stosują spójne konwencje: zmienne wejściowe z prefiksem `in_`, pełne nazwy modułów (FQCN) oraz walidację konfiguracji przed zastosowaniem zmian.
